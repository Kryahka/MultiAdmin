Условия использования

Последнее обновление: май 2025  ·  Версия 1.0

1

Принятие условий

Используя бота MultiAdmin в Discord, вы подтверждаете, что прочитали, поняли и соглашаетесь соблюдать настоящие Условия использования. Если вы не согласны с какими-либо пунктами — пожалуйста, прекратите использование бота.

2

Описание сервиса

MultiAdmin — Discord-бот с поддержкой искусственного интеллекта. Бот предоставляет следующие возможности:

  • Общение с AI через текстовые сообщения и изображения
  • Анализ изображений и GIF-файлов
  • Выдача случайных ссылок из Википедии
  • Управление настройками и правами доступа
  • Самомодификация кода по запросу владельца
3

Допустимое использование

Пользователь обязуется не использовать бота для:

  • Рассылки спама, флуда или нежелательного контента
  • Генерации незаконного, вредоносного или оскорбительного материала
  • Попыток взлома, обхода системы безопасности или несанкционированного доступа
  • Нарушения Условий использования Discord
  • Любых действий, нарушающих законодательство вашей страны
4

Ответственность за контент

Ответы бота генерируются сторонними AI-сервисами (Google Gemini, Groq, Mistral). Владелец бота не несёт ответственности за точность, актуальность или полноту предоставленной информации.

⚠️ Не используйте ответы бота как профессиональные консультации — юридические, медицинские, финансовые или иные.
5

Доступ и ограничения

Владелец бота вправе в любой момент:

  • Ограничить или отозвать доступ любого пользователя без объяснения причин
  • Изменить функциональность или прекратить работу бота
  • Обновить настоящие Условия без предварительного уведомления
6

Сторонние сервисы

Бот использует API следующих провайдеров для обработки запросов:

  • Google Gemini API — модели Gemini
  • Groq API — модели Llama
  • Mistral AI API — модели Mistral
  • Wikipedia API — случайные статьи

Использование этих сервисов регулируется их собственными условиями и политиками конфиденциальности.

7

Изменение условий

Мы оставляем за собой право изменять настоящие Условия в любое время. Продолжение использования бота после публикации изменений означает ваше согласие с новой редакцией.
