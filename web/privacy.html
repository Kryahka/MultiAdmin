Общие положения

Настоящая Политика конфиденциальности описывает, какие данные собирает и обрабатывает бот MultiAdmin, как они используются и какие права есть у пользователей в отношении своих данных.

Мы придерживаемся принципа минимально необходимой обработки данных и не продаём информацию о пользователях третьим лицам.
📦

Собираемые данные

В процессе работы бот обрабатывает следующие категории данных:

Тип данных Цель Хранение
Discord User ID Проверка прав доступа config.json (локально)
Discord Role ID Управление разрешениями config.json (локально)
Discord Guild ID Серверные настройки прав config.json (локально)
Текст сообщений Передача запроса в AI Только в памяти (до 20 пар)
Вложения (фото, GIF) Анализ AI при наличии в сообщении Не сохраняются
🔄

Обработка сообщений

Текстовые сообщения и вложения, адресованные боту, временно хранятся в оперативной памяти в рамках контекста текущего диалога (не более 20 сообщений). После перезапуска бота история полностью очищается.

Хранение в памяти
До 20 сообщений, сбрасывается при рестарте
Запись на диск
Сообщения не записываются. Только ID в config.json
🌐

Передача данных третьим сторонам

Для генерации ответов сообщения пользователя передаются одному из следующих AI-провайдеров:

  • Google Gemini — Политика конфиденциальности Google
  • Groq — Политика конфиденциальности Groq
  • Mistral AI — Политика конфиденциальности Mistral

Каждый провайдер обрабатывает данные согласно собственной политике. Рекомендуем ознакомиться с их документацией.

🛡️

Безопасность данных

Мы принимаем разумные меры для защиты данных, включая:

  • Хранение токенов и ключей API в переменных окружения (.env), не в коде
  • Разграничение доступа к функциям бота через систему разрешений
  • Ограничение операций с файлами пределами директории бота
👤

Права пользователей

В отношении своих данных, хранящихся в config.json, вы вправе:

  • Запросить информацию о том, какие данные хранятся
  • Потребовать удаления своих ID из конфигурации
  • Отозвать согласие, прекратив использование бота
Для реализации своих прав свяжитесь с владельцем бота напрямую через Discord.
📝

Изменение политики

Настоящая Политика конфиденциальности может быть обновлена в любое время. Актуальная версия всегда доступна по данной ссылке. Продолжение использования бота после обновления означает согласие с изменениями.
